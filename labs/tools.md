# Tools Guide

## Required Tools

| Tool | Purpose |
| --- | --- |
| Microsoft Learn | Study guide, modules, and sandbox learning. |
| Azure portal | Create and review Azure AI resources. |
| Azure Cloud Shell | Run Azure CLI commands. |
| Azure AI Foundry | Explore generative AI models and AI app development concepts. |
| Azure Machine Learning studio | Review automated ML, datasets, jobs, models, and endpoints. |

## Optional Tools

| Tool | Purpose |
| --- | --- |
| Visual Studio Code | Notes, JSON examples, and simple scripts. |
| diagrams.net | AI solution diagrams. |
| Spreadsheet tool | Workload mapping and comparison tables. |

## Cleanup Checklist

At the end of class, check for:

- Azure AI multi-service resources.
- Azure Machine Learning workspaces.
- Storage accounts created for labs.
- Model deployments or endpoints.
- Resource groups created only for training.
