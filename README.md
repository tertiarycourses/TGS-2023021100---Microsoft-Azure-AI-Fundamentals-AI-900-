# TGS-2023021100 - Microsoft Azure AI Fundamentals (AI-900)

> Course: Microsoft Azure AI Fundamentals (AI-900)  
> Course Code: TGS-2023021100

Hands-on Azure AI Fundamentals labs for learners building foundational knowledge of artificial intelligence, machine learning, and Microsoft Azure AI services. The labs cover AI workloads, responsible AI, machine learning concepts, Azure Machine Learning, computer vision, optical character recognition, natural language processing, speech, translation, document intelligence, generative AI, Azure AI Foundry, Azure OpenAI, and course review.

## How to Use These Labs

1. Read the learner guide before starting the first lab.
2. Complete the labs in order because later exercises reuse the same Azure AI vocabulary.
3. Use a Microsoft Learn sandbox, Azure free account, or instructor-provided Azure subscription.
4. Clean up any Azure resources created during class.
5. Review checkpoint questions after each lab.
